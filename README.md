# 💇‍♀️ Salon Booking System

A simple and modern web-based salon appointment booking system.  
Users can book appointments, select services, and view booking details through a dashboard.

---

## 🚀 Features

- 📅 Book salon appointments
- 💇 Select services (Hair Cut, Facial, Massage, etc.)
- 📱 Responsive and clean UI
- 📊 Dashboard to view booking details
- ⚡ Real-time form validation
- 🗂 Simple backend with JSON data storage

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Node.js (Backend)
- Express.js
- JSON (Data storage)
